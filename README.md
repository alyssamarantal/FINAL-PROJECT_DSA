School Equipment Borrowing System

Project Overview
The School Equipment Borrowing System is a console-based application written in C that manages the borrowing and returning of school equipment.

Its purpose is to help organize equipment inventory, track borrowers, process returns, and maintain borrowing records efficiently.

The system allows users to:

* Add new equipment
* Display available equipment
* Borrow equipment
* Return borrowed equipment
* Search for equipment
* Sort equipment records
* Display borrowing queue
* Edit equipment details
* Delete equipment records
* Filter and search equipment by category

Purpose of the Project

This project was developed to provide a simple and efficient way of managing school equipment borrowing transactions.

It helps:
* Track available equipment
* Prioritize borrowers
* Maintain borrowing history
* Prevent inventory errors
* Organize equipment records

Data Structures Used

1. Array
The program uses an array to store all equipment records.

2. Linked List
The program uses a singly linked list to manage the borrowing queue.

3. Priority Queue
The linked list is implemented as a priority queue.

Algorithms Used

1. Linear Search
Used to search for equipment by name.

2. Bubble Sort
Used to sort equipment alphabetically.

3. Queue Insertion Algorithm
Used to insert borrowers according to priority.

4. Input Validation
The program validates equipment names.
